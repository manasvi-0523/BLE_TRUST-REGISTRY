# BLE Trust Registry

BLE Trust Registry is a professional real-time BLE monitoring system for trusted device baselines, anomaly detection, and tamper-evident incident logging.

The active application lives in:

```text
ble-trust-registry/
```

Start there for installation, architecture diagrams, workflow diagrams, and run commands:

```text
ble-trust-registry/README.md
```

## Repository Map

```text
BLE_TRUST-REGISTRY/
```

## Fast Start

```powershell
cd ble-trust-registry
.\scripts\start-dev.cmd
```

Open:

```text
http://localhost:3000
```

Backend status:

```text
http://127.0.0.1:8000/status
```
